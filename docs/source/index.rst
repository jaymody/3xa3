Lines Per Minute Documentation
==============================

Lines Per Minute (lpm) is a command-line typing practice tool made for programmers. Inspired by `github.com/cslarsen/wpm <link https://github.com/cslarsen/wpm>`_.

**Installation:**

``pip install lpm``
